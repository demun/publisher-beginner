# Comments
주석 태그는 HTML 소스 코드에 주석을 삽입하는 데 사용됩니다.  
주석은 브라우져에서 보여지지 않지만 소스코드에는 보여집니다.

코드예:
```html
<!-- 이곳은 주석입니다. -->
```

보여지는예:
<!-- 이곳은 주석입니다. -->


# 조건부 주석

IE 특정버젼에만 적용되는 주석을 사용할 수 있습니다.

아래 코드는 IE 9버젼에만 보여지고 다른 브라우져에서는 안보입니다.

```html
<!--[if IE 9]>
  <div>이 글은 IE 9에서만 보입니다.</div>
<![endif]-->
```
