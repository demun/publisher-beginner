# Paragraphs

HTML `<p>` 요소는 단락을 정의합니다 

!!! tip
    브라우저는 단락 앞뒤에 공백을 자동으로 추가합니다 (여백).

`<p>` 태그안에서 줄바꿈을 하여도 브라우져는 무시한다

예:
```html
<p>
This paragraph
contains a lot of lines
in the source code,
but the browser 
ignores it.
</p>
```

보여지는 예:
```html
This paragraph contains a lot of lines in the source code, but the browser ignores it.
```
